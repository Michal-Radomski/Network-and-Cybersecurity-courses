The **Caesar Cipher** is one of the oldest and simplest encryption techniques. It is a type of **substitution cipher** where
each letter in the original text (plaintext) is shifted a fixed number of places down the alphabet.

For example, with a shift of **3**:

- `A` becomes `D`
- `B` becomes `E`
- `X` becomes `A` (wrapping around to the beginning)

---

## TypeScript Implementation in Node.js

Here is a complete TypeScript implementation that handles both encryption and decryption, preserving the case (upper/lower)
of the letters while leaving spaces and punctuation unchanged.

### `caesar.ts`

```typescript
/**
 * Encrypts or decrypts text using the Caesar Cipher.
 * @param text The input string to process.
 * @param shift The number of positions to shift the alphabet (can be negative for decryption).
 */
export function caesarCipher(text: string, shift: number): string {
  // Handle negative shifts and shifts greater than 26
  const normalizedShift = ((shift % 26) + 26) % 26;

  return text
    .split("")
    .map((char) => {
      const code = char.charCodeAt(0);

      // Uppercase letters (A-Z: 65-90)
      if (code >= 65 && code <= 90) {
        return String.fromCharCode(((code - 65 + normalizedShift) % 26) + 65);
      }

      // Lowercase letters (a-z: 97-122)
      else if (code >= 97 && code <= 122) {
        return String.fromCharCode(((code - 97 + normalizedShift) % 26) + 97);
      }

      // Non-alphabetic characters remain unchanged
      return char;
    })
    .join("");
}

// Example usage:
const secretMessage = "Hello, World! 2026";
const shiftAmount = 5;

// 1. Encrypt
const encrypted = caesarCipher(secretMessage, shiftAmount);
console.log(`Encrypted: ${encrypted}`);
// Output: "Mjqqt, Btwqi! 2026"

// 2. Decrypt (Shift in the opposite direction)
const decrypted = caesarCipher(encrypted, -shiftAmount);
console.log(`Decrypted: ${decrypted}`);
// Output: "Hello, World! 2026"
```

### How the Math Works

To wrap the alphabet cleanly without breaking when it hits `Z`, the code converts a character to a 0-25 scale (e.g.,
`code - 65` for uppercase), adds the shift, applies a modulo 26 (`% 26`), and then adds the base code back.
