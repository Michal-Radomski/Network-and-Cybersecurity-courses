Generating a key pair using OpenSSL on Ubuntu is a straightforward process. You will typically generate a private key first,
and then derive the public key from it.

Here is the quickest and most secure way to do it using standard RSA keys.

---

## Step 1: Open Your Terminal

You can open the terminal on Ubuntu by pressing `Ctrl + Alt + T`.

## Step 2: Generate the Private Key

Run the following command to generate a secure, 4096-bit RSA private key:

```bash
openssl genpkey -algorithm RSA -out private_key.pem -pkeyopt rsa_keygen_bits:4096

```

- **`private_key.pem`**: This is the name of the file where your private key will be saved.
- **`4096`**: This specifies the key length. 4096-bit is highly secure and standard practice.

> ⚠️ **Important:** Keep your private key secret. Anyone with access to this file can impersonate you or decrypt your data.

## Step 3: Extract the Public Key

Now, use the private key you just created to generate its corresponding public key:

```bash
openssl pkey -in private_key.pem -pubout -out public_key.pem

```

- **`-in private_key.pem`**: Tells OpenSSL to use your existing private key as the input.
- **`-pubout`**: Specifies that you want to output a public key.
- **`public_key.pem`**: This is the file where your public key will be saved. You can safely share this file with anyone.

---

## Verifying Your Keys (Optional)

If you want to view the actual text contents of your newly generated keys directly in the terminal, you can use the `cat`
command:

- To see the private key: `cat private_key.pem`
- To see the public key: `cat public_key.pem`
