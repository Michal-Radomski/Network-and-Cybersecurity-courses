To use BitTorrent on Ubuntu, you can use the default built-in torrent client, **Transmission**, or install alternative
clients depending on whether you prefer a graphical interface or a command-line tool.

---

## Method 1: Using Transmission (Default Desktop GUI)

Ubuntu comes with **Transmission** pre-installed on most desktop releases.

### Step 1: Open Transmission

1. Press the `Super` key (Windows key) to open the Application Menu.
2. Type **Transmission** and launch the app.

### Step 2: Add a Torrent

- **Via Magnet Links:** Click a `magnet:` link in your web browser. The browser will prompt you to open the link with
  Transmission automatically.
- **Via `.torrent` Files:** 1. Download the `.torrent` file (for example, an official Ubuntu Desktop ISO torrent).

2. In Transmission, click the **File** menu $\rightarrow$ **Open**, or click the **+** (Add) icon in the top toolbar.
3. Select your `.torrent` file and click **Open**.

### Step 3: Manage Downloads

- Choose the destination folder where you want to save the downloaded files.
- Click **Add** to begin the download.
- You can pause, resume, or limit upload/download speeds by right-clicking any active transfer.

---

## Method 2: Installing Alternative GUI Clients

If Transmission is not installed or you prefer a different interface, you can install popular clients like **qBittorrent** or
**Deluge**.

### Installing qBittorrent via Terminal

Open your terminal (`Ctrl` + `Alt` + `T`) and run:

```bash
sudo apt update
sudo apt install qbittorrent

```

Once installed, launch it from your application menu or by running `qbittorrent` in the terminal.

---

## Method 3: Command-Line Client (`aria2` or `transmission-cli`)

For headless servers or lightweight terminal usage, you can use `aria2` or `transmission-cli`.

### Using `aria2`

1. Install `aria2`:

```bash
sudo apt update
sudo apt install aria2

```

2. Download using a `.torrent` file:

```bash
aria2c /path/to/file.torrent

```

3. Download using a magnet link (wrap the link in quotes):

```bash
aria2c "magnet:?xt=urn:btih:EXAMPLE_HASH_HERE..."

```
