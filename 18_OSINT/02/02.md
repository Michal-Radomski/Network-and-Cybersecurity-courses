## Anatomy of the Link

A magnet link breaks down into several key components:

- **`magnet:?`**: Identifies the scheme telling software (like BitTorrent clients) how to handle the link.
- **`xt=urn:btih:`**: Stands for _"exact topic = uniform resource name: BitTorrent info-hash"_.
- **`0595273ab674e05131a757f69f494a4285b429aa`**: The unique SHA-1 cryptographic hash of the content. This serves as the
  content's digital fingerprint so peers know exactly which files to request.
- **`dn=`**: Stands for _"display name"_. It gives a human-readable title to the download:
  `Facebook Leak [2019][533M Records][106 Countries]`.
