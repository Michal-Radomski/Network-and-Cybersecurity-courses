Depending on how deeply you want to check, you can find out which version of TLS a website is using by looking at your
browser, running a command in your terminal, or using a quick online tool.

Here are the simplest ways to do it:

### Method 1: Use Your Web Browser (Easiest)

You can check the specific TLS version your browser negotiated with the website.

- **Google Chrome / Microsoft Edge:**

1. Go to the website.
2. Click the **Lock icon** (or tune/settings icon) next to the URL in the address bar.
3. Click **Connection is secure** (or **Connection settings**).
4. You will see a quick summary of the encryption, for example: _"The connection to this site is encrypted and authenticated
   using TLS 1.3..."_

- **Mozilla Firefox:**

1. Go to the website.
2. Click the **Lock icon** to the left of the URL.
3. Click **Connection secure**, then click **More Information**.
4. Under the **Technical Details** section of the Security tab, it will list the TLS version and cipher suite.

---

### Method 2: Use Online Scan Tools (Most Detailed)

If you want to know **all** versions of TLS a website supports (not just the one your browser chose), online checkers are
your best option.

- **Qualys SSL Labs (SSL Server Test):** Go to [ssllabs.com/ssltest](https://www.ssllabs.com/ssltest/), type in the domain
  name, and hit submit. Scroll down to the **Configuration** section to see exactly which versions (TLS 1.0, 1.1, 1.2, 1.3)
  are enabled or disabled.
- **CDN77 TLS Test:** A much faster alternative for a quick read is [cdn77.com/tls-test](https://www.cdn77.com/tls-test).

---

### Method 3: Use the Command Line (For Developers/Techies)

If you have a terminal handy, you can test for specific TLS versions using `openssl` or `curl`.

**Using OpenSSL:** Run this command to check if a site supports a specific version (like TLS 1.3). If it connects
successfully, it's supported:

```bash
openssl s_client -connect google.com:443 -tls1_3

```

_(Swap `-tls1_3` with `-tls1_2` to check for TLS 1.2)._

**Using Curl:** You can ask `curl` to fetch the site using a specific protocol version and print the connection handshake
details:

```bash
curl -I -v --tlsv1.3 https://google.com

```

Look for a line in the output that says something like `SSL connection using TLSv1.3`.
