In web development, a **fragment** (often called a **fragment identifier** or a **hash**) is used to jump directly to a
specific section of a webpage. It acts like a bookmark.

Here is exactly how you create a fragment in your HTML and how you target it using a URL.

---

## 1. How to create a fragment in HTML

To make a section of your webpage "linkable," you need to give it a unique identifier using the **`id` attribute** on an HTML
element.

```html
<h2 id="features-section">Our Amazing Features</h2>
<p>Here is a list of all the cool things our app can do...</p>
```

> **Rule:** The `id` must be completely unique on that page. It shouldn't contain spaces (use hyphens or underscores
> instead).

---

## 2. How to link to that fragment in a URL

To link directly to that specific section, you append a **hash symbol (`#`)** followed by the `id` name to the end of the
URL.

### From an external website:

If someone is visiting your site from somewhere else on the internet, the URL looks like this:
`https://www.example.com/about.html#features-section`

### From the same webpage (Anchor Link):

If you want to create a menu or a "Back to Top" button on the exact same page, you can omit the web address and just use the
fragment in your HTML link (`<a>`) tag:

```html
<a href="#features-section">Jump to Features</a>
```

---

## A Complete Live Example

Here is how a simple "Table of Contents" page looks when putting HTML fragments and URLs together:

```html
<!DOCTYPE html>
<html>
  <head>
    <title>My Document</title>
  </head>
  <body>
    <h1>Table of Contents</h1>
    <ul>
      <li><a href="#chapter1">Go to Chapter 1</a></li>
      <li><a href="#chapter2">Go to Chapter 2</a></li>
    </ul>

    <div style="margin-top: 1000px;"></div>

    <h2 id="chapter1">Chapter 1: The Beginning</h2>
    <p>Once upon a time...</p>

    <div style="margin-top: 1000px;"></div>

    <h2 id="chapter2">Chapter 2: The Adventure</h2>
    <p>And then things got interesting...</p>
  </body>
</html>
```
