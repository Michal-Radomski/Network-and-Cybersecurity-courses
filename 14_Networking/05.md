If a bit is a single light switch, and a byte is a group of 8 switches used to represent a character, **Unicode** is the
universal "decoder ring" that tells computers exactly what character those bits and bytes actually mean.

Put simply, Unicode is the international standard that assigns a unique number to every single character, letter, symbol, and
emoji used across the digital world.

---

## The Problem Unicode Solved

In the early days of computing, different countries and companies used different systems to translate numbers into text. The
most famous was **ASCII** (American Standard Code for Information Interchange).

ASCII worked great for English because it used 7 bits to represent 128 characters—enough for the English alphabet, numbers,
and basic punctuation.

However, as computers went global, chaos ensued:

- A system designed for French might interpret the binary code `130` as **é**.
- A system designed for Hebrew might interpret that exact same code `130` as the letter **ג (Gimel)**.
- If you sent an email from one system to the other, the text would turn into unreadable garbage (often called _Mojibake_).

---

## How Unicode Works

Unicode changed everything by creating one massive, standardized list. Instead of forcing everyone to share 128 or 256 slots,
Unicode provides a unique **Code Point** (written like `U+0041`) for every character in existence.

It covers:

- Every modern language (English, Chinese, Arabic, Hindi, etc.)
- Historical and dead languages (Egyptian Hieroglyphs, Phoenician)
- Mathematical symbols, currency signs, and yes—**all emojis**.

> **Example:** > \* The letter **A** is always **U+0041**
>
> - The Euro symbol **€** is always **U+20AC**
> - The "Grinning Face" emoji 😀 is always **U+1F600**

Because every device in the world agrees on this master list, a text message sent from a phone in Tokyo will look exactly the
same when opened on a laptop in New York.

---

## Unicode vs. UTF-8 (How it's stored)

People often confuse **Unicode** with **UTF-8**. Here is the difference:

- **Unicode is the List:** It is just the dictionary mapping characters to numbers (e.g., Code Point `U+1F600` = 😀).
- **UTF-8 is the Code:** It is the _encoding system_ that translates those code point numbers into actual bits and bytes so a
  computer's hard drive can store them efficiently.

UTF-8 is brilliant because it is flexible. For standard English text, it only uses 1 byte per character (saving space). For
complex symbols or emojis, it automatically scales up to use 2, 3, or 4 bytes. Today, UTF-8 is used by over 98% of all
websites.
