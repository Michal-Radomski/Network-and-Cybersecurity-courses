Just like cars use different engines under the hood, web browsers use **browser engines** (also called rendering engines) to
take HTML, CSS, and JavaScript code and turn it into the visual webpage you see on your screen.

Over the years, the browser market has consolidated heavily. Today, there are really only **three major engines** that power
almost every browser on earth.

---

### 1. Blink (The Market Dominant Engine)

Developed by Google as part of the open-source Chromium project, Blink was originally forked from WebKit in 2013. It is by
far the most widely used engine today.

- **Google Chrome**
- **Microsoft Edge**
- **Opera**
- **Brave**
- **Vivaldi**
- **Samsung Internet** (on Android phones)

---

### 2. WebKit (The Apple Standard)

Originally created by Apple for Safari (basing it on an older open-source engine called KHTML).

- **Apple Safari** (macOS, iOS, iPadOS)
- **Every single browser on iOS/iPhone:** Apple historically forced a strict rule on the App Store: _all_ browsers running on
  iOS (including Chrome, Firefox, and Edge for iPhone) **must** use WebKit under the hood instead of their native engines.
  _(Note: Regulatory changes, like the DMA in the EU, have recently begun forcing Apple to allow other engines on iOS, but
  WebKit remains the default standard for Apple ecosystems)._

---

### 3. Gecko (The Independent Engine)

Developed by the Mozilla Foundation. It was built from scratch to be highly customizable and standards-compliant, ensuring
that the web didn't become a monopoly controlled by a single company.

- **Mozilla Firefox**
- **Waterfox / LibreWolf** (privacy-focused Firefox forks)
- **Tor Browser** (the anonymity-focused browser uses a modified version of Gecko)

---

### 💡 Quick Summary: The JavaScript Engines

While the _rendering_ engines above handle the layout and visuals, they pair up with specialized **JavaScript engines** to
run the actual code and web apps. They usually travel in pairs:

| Rendering Engine | Paired JavaScript Engine     | Main Browser        |
| ---------------- | ---------------------------- | ------------------- |
| **Blink**        | **V8** (Developed by Google) | Chrome, Edge, Brave |
| **WebKit**       | **JavaScriptCore** (Nitro)   | Safari              |
| **Gecko**        | **SpiderMonkey**             | Firefox             |
