Network ports act as communication channels for different services and protocols over a network. The Internet Assigned
Numbers Authority (IANA) categorizes them into ranges, with **Well-Known Ports (0–1023)** being the most common.

Here is a breakdown of the most frequently used ports organized by their category:

## Web Traffic

| Port Number         | Transport Protocol | Service / Protocol Name                | Description                                                                |
| ------------------- | ------------------ | -------------------------------------- | -------------------------------------------------------------------------- |
| **80**              | TCP                | **HTTP** (Hypertext Transfer Protocol) | Unencrypted web traffic.                                                   |
| **443**             | TCP / UDP (QUIC)   | **HTTPS** (HTTP Secure)                | Encrypted web traffic using TLS/SSL.                                       |
| **8080** / **8443** | TCP                | **HTTP-Alt / HTTPS-Alt**               | Commonly used for proxy servers, QA testing, or internal web applications. |

---

## File Transfer & Remote Access

| Port Number | Transport Protocol | Service / Protocol Name           | Description                                                                |
| ----------- | ------------------ | --------------------------------- | -------------------------------------------------------------------------- |
| **20 / 21** | TCP                | **FTP** (File Transfer Protocol)  | Port 20 is for data transfer; Port 21 is for command control. Unencrypted. |
| **22**      | TCP                | **SSH / SFTP** (Secure Shell)     | Secure remote command-line access and encrypted file transfers.            |
| **23**      | TCP                | **Telnet**                        | Legacy remote login service. Highly insecure (sent in plaintext).          |
| **3389**    | TCP / UDP          | **RDP** (Remote Desktop Protocol) | Microsoft's proprietary protocol for graphical remote management.          |

---

## Email Services

| Port Number | Transport Protocol | Service / Protocol Name                     | Description                                                      |
| ----------- | ------------------ | ------------------------------------------- | ---------------------------------------------------------------- |
| **25**      | TCP                | **SMTP** (Simple Mail Transfer Protocol)    | Historically used for routing email between mail servers.        |
| **587**     | TCP                | **SMTP Secure (Submission)**                | Modern, encrypted standard for users to send outgoing email.     |
| **110**     | TCP                | **POP3** (Post Office Protocol v3)          | Downloading email from a server to a local client (unencrypted). |
| **995**     | TCP                | **POP3S** (POP3 Secure)                     | Encrypted POP3 using TLS/SSL.                                    |
| **143**     | TCP                | **IMAP** (Internet Message Access Protocol) | Managing and syncing email directly on a server (unencrypted).   |
| **993**     | TCP                | **IMAPS** (IMAP Secure)                     | Encrypted IMAP using TLS/SSL.                                    |

---

## Core Network Services

| Port Number   | Transport Protocol | Service / Protocol Name                        | Description                                                                              |
| ------------- | ------------------ | ---------------------------------------------- | ---------------------------------------------------------------------------------------- |
| **53**        | UDP / TCP          | **DNS** (Domain Name System)                   | Translates human-readable domain names (like google.com) to IP addresses.                |
| **67 / 68**   | UDP                | **DHCP** (Dynamic Host Configuration Protocol) | Automatically assigns IP addresses to devices on a network (67 is server, 68 is client). |
| **123**       | UDP                | **NTP** (Network Time Protocol)                | Synchronizes computer clocks across a network.                                           |
| **161 / 162** | UDP                | **SNMP** (Simple Network Management Protocol)  | Monitoring and managing network devices (routers, switches).                             |

---

## Databases

| Port Number | Transport Protocol | Service / Protocol Name  | Description                                                |
| ----------- | ------------------ | ------------------------ | ---------------------------------------------------------- |
| **1433**    | TCP                | **Microsoft SQL Server** | Default port for MSSQL database connections.               |
| **1521**    | TCP                | **Oracle DB**            | Default port for Oracle database listener.                 |
| **3306**    | TCP                | **MySQL**                | Default port for MySQL database engines.                   |
| **5432**    | TCP                | **PostgreSQL**           | Default port for PostgreSQL database connections.          |
| **27017**   | TCP                | **MongoDB**              | Default port for MongoDB instances.                        |
| **6379**    | TCP                | **Redis**                | Default port for the Redis in-memory cache and data store. |

---

## Directory & File Sharing (LAN)

| Port Number         | Transport Protocol | Service / Protocol Name        | Description                                                                    |
| ------------------- | ------------------ | ------------------------------ | ------------------------------------------------------------------------------ |
| **389** / **636**   | TCP / UDP          | **LDAP / LDAPS**               | Lightweight Directory Access Protocol (636 is the encrypted SSL/TLS version).  |
| **445**             | TCP                | **SMB** (Server Message Block) | Used for sharing files and printers on Windows-based local networks.           |
| **137 / 138 / 139** | UDP / TCP          | **NetBIOS**                    | Legacy Windows networking services for name resolution and session management. |
