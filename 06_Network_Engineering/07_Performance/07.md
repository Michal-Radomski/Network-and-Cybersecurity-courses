To run an Express.js server specifically on IPv6 (`::1`) on your localhost and automatically open the browser, you can use
Node's built-in `http` configurations alongside a popular, lightweight npm package called `open`.

Here is a step-by-step guide to setting this up.

### 1. Initialize and Install Dependencies

First, make sure you have your project folder set up and the necessary packages installed. You will need `express` and
`open`.

```bash
mkdir express-ipv6-demo
cd express-ipv6-demo
npm init -y
npm install express open

```

### 2. Create the Server Code

Create a file named `server.js` and add the following code.

To force Express to use IPv6, you pass the IPv6 loopback address `::` (which binds to all IPv6 addresses on the machine) or
`::1` (pure localhost IPv6) as the second argument to the `.listen()` method.

```javascript
import express from "express";
import open from "open";

const app = express();
const PORT = 3000;
// '::' binds to all IPv6 addresses, or use '::1' for strict IPv6 localhost
const HOST = "::1";

app.get("/", (req, res) => {
  res.send("<h1>Hello from Express on IPv6!</h1>");
});

const server = app.listen(PORT, HOST, async () => {
  // Format the URL correctly for IPv6 (requires square brackets)
  const url = `http://[${HOST}]:${PORT}`;

  console.log(`Server is running successfully at ${url}`);

  // Automatically open the browser
  await open(url);
});
```

> 💡 **Note on ES Modules:** The code above uses `import` syntax. Ensure you have `"type": "module"` added to your
> `package.json` file, or switch the imports to standard `require()` syntax if you are using CommonJS.

### 3. Run the Server

Execute the script using Node.js:

```bash
node server.js

```

### How it Works

- **IPv6 Formatting:** In web browsers, IPv6 addresses must be enclosed in square brackets, which is why the `open` URL is
  formatted as `http://[::1]:3000`.
- **The `open` Package:** The `open` library detects your operating system's default browser and safely spawns a process to
  open the generated URL the moment the server successfully spins up.
