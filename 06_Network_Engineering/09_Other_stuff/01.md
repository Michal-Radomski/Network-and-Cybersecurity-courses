Wireshark is a free, open-source network protocol analyzer used to capture and inspect network traffic in detail. It helps
with troubleshooting, security analysis, protocol debugging, and learning how networks work.
[wireshark](https://www.wireshark.org/docs/wsug_html_chunked/ChapterIntroduction.html)

## What it does

Wireshark can capture live packets from a network interface, open saved capture files, filter traffic by many criteria, and
show detailed protocol information in a three-pane interface. It supports live capture and offline analysis, and it runs on
major platforms like Windows, Linux, and macOS.
[wireshark](https://www.wireshark.org/docs/wsug_html_chunked/ChapterIntroduction.html)

## Common uses

People use Wireshark to find network problems, verify application behavior, inspect security events, and study how protocols
behave at the packet level. It is especially useful when you need to see exactly what data is moving across the network
rather than just knowing that a connection failed. [techtarget](https://www.techtarget.com/whatis/definition/Wireshark)

## Limits

Wireshark observes traffic but does not actively fix problems or act as an intrusion detection system. It is a diagnostic and
analysis tool, not a security appliance.
[wireshark](https://www.wireshark.org/docs/wsug_html_chunked/ChapterIntroduction.html)
