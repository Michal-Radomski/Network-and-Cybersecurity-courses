It is entirely possible to add a self-signed TLS certificate to both Ubuntu system-wide and to Firefox specifically. Since
Firefox uses its own independent certificate store, you usually have to add it to both places if you want both command-line
tools (like `curl`) and your browser to trust the certificate.

Here is how to do both.

---

## 1. Add the Certificate to Ubuntu (System-Wide)

Adding the certificate here ensures that system utilities, Python scripts, `curl`, and `wget` will trust your self-signed
certificate.

### Steps:

1. **Copy the certificate** (it should be in `.crt` format) to the system's shared certificates directory. Open your terminal
   and run:

```bash
sudo cp my-certificate.crt /usr/local/share/ca-certificates/

```

2. **Update the certificate store** by running the following command:

```bash
sudo update-ca-certificates

```

3. **Verify:** You should see output indicating that a certificate was added. You can test it by running a `curl` command
   against your local HTTPS server.

---

## 2. Add the Certificate to Firefox on Ubuntu

Firefox ignores the Ubuntu system certificate store by default and uses its own built-in database.

### Method A: The Graphical Interface (Easiest)

1. Open **Firefox**.
2. Click the menu button (three horizontal lines) and select **Settings**.
3. In the left sidebar, click **Privacy & Security**.
4. Scroll all the way down to the **Certificates** section and click **View Certificates...**.
5. In the window that pops up, select the **Authorities** tab.
6. Click **Import...** at the bottom.
7. Browse to your `.crt` or `.pem` file and select it.
8. A box will appear asking you to trust the certificate. Check the box for **"Trust this CA to identify websites"** and
   click **OK**.

### Method B: Make Firefox Trust the Ubuntu Store

If you are using a standard Ubuntu installation, Firefox is installed as a **Snap package**. You can actually force Firefox
to look at your Ubuntu system certificates automatically so you don't have to import them manually.

1. Open Firefox and type `about:config` into the address bar, then press Enter.
2. Accept the warning layout.
3. Search for the setting: `security.enterprise_roots.enabled`
4. Double-click it to change its value from `false` to **`true`**.
5. Restart Firefox. It will now trust any certificate you added via the `update-ca-certificates` command in Step 1.
