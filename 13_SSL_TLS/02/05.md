Hashing a file on Ubuntu is incredibly straightforward because the necessary tools are built right into the system terminal.
You don’t need to install any extra software.

Here is how to do it step-by-step using the most secure and common standard, **SHA-256**, as well as other formats.

---

## Step 1: Open the Terminal

You can quickly open the terminal by pressing **`Ctrl` + `Alt` + `T**` on your keyboard.

---

## Step 2: Navigate to Your File (Optional)

If your file is in your `Downloads` or `Documents` folder, use the `cd` command to go there. For example:

```bash
cd ~/Downloads

```

---

## Step 3: Run the Hashing Command

### 1. To get a SHA-256 Hash (Recommended)

Type `sha256sum` followed by the exact name of your file:

```bash
sha256sum filename.txt

```

**The Output will look like this:**

```text
e3b0c44298fc1c149afbf4c8996fb92427ae41e4649b934ca495991b7852b855  filename.txt

```

The long string of letters and numbers is your file's unique SHA-256 digital fingerprint.

### 2. To get a SHA-512 Hash (For maximum security)

If you need a longer, even more secure hash, use:

```bash
sha512sum filename.txt

```

### 3. To get an MD5 Hash (Legacy/Basic integrity check)

If you are downloading an older file or a Linux ISO that explicitly asks you to verify an MD5 checksum:

```bash
md5sum filename.txt

```

---

## Pro-Tip: Automatically Verifying a File

If you downloaded a large file (like an Ubuntu installation `.iso`) alongside a text file containing its official hashes
(usually named `SHA256SUMS`), you don't have to manually match the long strings with your eyes. Ubuntu can do it for you.

1. Put the downloaded file and the `SHA256SUMS` file in the same folder.
2. Run this command:

```bash
sha256sum -c SHA256SUMS

```

3. If the file is perfect and matches, the terminal will print:

```text
ubuntu-desktop.iso: OK

```
