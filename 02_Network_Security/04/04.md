Advanced Encryption Package (AEP) is a commercial file encryption software specifically for Windows, designed to secure files
and folders using a wide range of cryptographic algorithms. [1, 2, 3, 4, 5]

## Key Features of AEP

- Algorithm Variety: Supports up to 20 different encryption algorithms, including industry standards like AES (Rijndael),
  Blowfish, Twofish, and Serpent.
- Batch Processing: Allows you to encrypt multiple files or entire directory trees simultaneously.
- Self-Decrypting Files (SFX): Can create executable files (.exe) that the recipient can decrypt without needing any
  encryption software installed.
- Secure File Deletion: Includes a built-in file shredder that overwrites original files beyond recovery, meeting various
  government standards like U.S. DoD 5200.28.
- Clipboard Encryption: Features a utility to encrypt or "scramble" text copied to the clipboard for secure transmission in
  emails or chats. [5, 6, 7, 8, 9]

## Linux Equivalents

Because AEP is a Windows-native application, Linux users typically use one of the following tools depending on their specific
needs: [10]

- [GnuPG (GPG)](https://www.gnupg.org/): The gold standard for Linux file encryption. Like AEP, it supports symmetric and
  asymmetric encryption and is pre-installed on most distributions.
- VeraCrypt: An open-source, multi-platform tool ideal if you need "on-the-fly" encryption. It can create encrypted virtual
  disks (containers) or encrypt entire partitions.
- 7-Zip (p7zip): A widely used archiving utility that provides AES-256 encryption for compressed files, similar to AEP's .ZIP
  support.
- Kryptor: A modern, cross-platform alternative specifically designed to be an easy-to-use replacement for tools like AEP and
  AxCrypt.
- [AES Crypt](https://www.aescrypt.com/linux_aes_crypt.html): A simple tool that provides a right-click "Open With" GUI
  option for encrypting files on Linux, similar to AEP’s Windows Explorer integration. [11, 12, 13, 14, 15, 16, 17]

| Windows Feature [17] | Closest Linux Equivalent                 |
| -------------------- | ---------------------------------------- |
| GUI Integration      | AES Crypt or Nautilus Encryption Utility |
| Multiple Algorithms  | GnuPG (GPG)                              |
| Encrypted Containers | VeraCrypt                                |
| CLI Automation       | GnuPG (GPG) or Openssl                   |
