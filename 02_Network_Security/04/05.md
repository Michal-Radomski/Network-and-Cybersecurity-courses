GnuPG (GPG) is a robust tool for secure communication and data storage, and it usually comes pre-installed on most modern
Ubuntu versions. [1, 2]

## Installation

If it is not already on your system, you can install it using the standard package manager: [1]

1.  Update your package list:

sudo apt update

2.  Install GnuPG:

sudo apt install gnupg

On older Ubuntu versions, you might need to specify sudo apt install gnupg2. [1, 2, 3, 4]

## 1. Generating Your Key Pair

Before encrypting files for others, you need to create your own "digital identity" consisting of a public key (for others to
encrypt data for you) and a private key (for you to decrypt it). [1, 5]

1.  Start the generation process:

gpg --full-generate-key

2.  Follow the prompts:

- Key Type: Choose the default RSA and RSA.
  - Key Size: Select 4096 bits for maximum security.
  - Expiration: Enter 0 if you don't want the key to ever expire.
  - User ID: Provide your real name and email address.
  - Passphrase: Choose a very strong, memorable password; if you forget this, you lose access to your encrypted data forever.
    [1, 5, 6, 7, 8]

## 2. Basic File Encryption

GPG offers two ways to encrypt files: Symmetric (password-based) and Asymmetric (key-based). [9, 10, 11, 12]

## Symmetric Encryption (Password only) [13]

This creates an encrypted file that can be opened by anyone who knows the password. [11]

- Command: gpg --symmetric filename.txt
- Result: Creates filename.txt.gpg. [14, 15, 16]

## Asymmetric Encryption (Using a Public Key) [17]

Use this to encrypt a file so that only a specific person (the recipient) can open it using their private key. [15]

- Command: gpg --encrypt --recipient recipient@email.com filename.txt. Note: You must first import the recipient's public key
  using gpg --import publickey.asc. [15, 18, 19]

## 3. Decrypting a File

To decrypt any .gpg file, the command is the same regardless of the encryption method: [20]

- Command: gpg --decrypt filename.txt.gpg
- Save to a specific file: gpg --output original.txt --decrypt filename.txt.gpg. [14, 21, 22]

## 4. Managing Your Keys

- List public keys: gpg --list-keys.
- List private keys: gpg --list-secret-keys.
- Export your public key: To let others send you encrypted files, export your key to a file to share with them:

gpg --armor --export your@email.com > mypublickey.asc
