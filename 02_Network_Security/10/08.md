In networking, ports are categorized by their usage. Well-known ports (0–1023) are reserved for standard system services,
while registered ports (1024–49151) are typically used by specific applications and databases. [1, 2, 3, 4]

## Web & Essential Services

These are the most frequently encountered ports during basic network scanning.

- 80 (TCP) - HTTP: Standard, unencrypted web traffic.
- 443 (TCP) - HTTPS: Secure, encrypted web traffic.
- 53 (UDP/TCP) - DNS: Resolves domain names to IP addresses.
- 67/68 (UDP) - DHCP: Automatically assigns IP addresses to devices on a network. [5, 6, 7, 8, 9]

## Remote Access & Management

Used by administrators to manage servers and devices remotely. [6]

- 22 (TCP) - SSH: Secure remote login and file transfer (SFTP); the modern standard for management.
- 23 (TCP) - Telnet: Older, unencrypted remote login. It is often targeted because it is insecure.
- 3389 (TCP) - RDP: Microsoft's Remote Desktop Protocol for Windows machines. [2, 6, 7, 8, 9, 10, 11]

## File Transfer & Sharing

- 20/21 (TCP) - FTP: Port 21 is for commands, while Port 20 handles the actual data transfer.
- 445 (TCP) - SMB: Used for Windows file and printer sharing on modern networks.
- 137/138/139 (UDP/TCP) - NetBIOS: Legacy Windows file sharing protocols. [1, 4, 5, 6, 10, 11]

## Email Protocols

- 25 (TCP) - SMTP: Used for sending emails between servers.
- 110 (TCP) - POP3: Used by clients to retrieve email from a server.
- 143 (TCP) - IMAP: Used to sync and manage email across multiple devices. [1, 6, 7, 9, 11]

## Popular Databases

Scanning these often reveals sensitive backend systems.

- 3306 (TCP) - MySQL: The most common open-source database port.
- 1433 (TCP) - MSSQL: Default port for Microsoft SQL Server.
- 5432 (TCP) - PostgreSQL: Frequently used in modern web applications.
- 27017 (TCP) - MongoDB: A popular NoSQL database port. [2, 3, 7, 11, 12, 13, 14]
