To use Nmap (Network Mapper), you follow a standard command structure: nmap [Scan Type] [Options] [Target]. It is highly
flexible, allowing you to scan everything from a single IP to an entire network range. [1, 2, 3, 4]

## 1. Basic Syntax & Installation

If it is not already installed on your Ubuntu system, you can get it via the terminal: [5]

- Install: sudo apt install nmap
- Simple Scan: nmap 192.168.1.1 (Scans the top 1,000 most common TCP ports on that specific IP) [2, 5, 6, 7]

## 2. Common Scan Types

Choose a scan type based on whether you need speed, stealth, or deep information: [8, 9, 10]

- Stealth SYN Scan (-sS): The most popular scan; it is fast and hard for some firewalls to detect because it doesn't complete
  the 3-way handshake.
- Service & Version Detection (-sV): Probes open ports to determine exactly what software and version is running (e.g.,
  Apache 2.4.41).
- OS Detection (-O): Analyzes the packet responses to "guess" the target's operating system.
- Aggressive Scan (-A): A "one-stop" flag that enables OS detection, version detection, script scanning, and traceroute. [11,
  12, 13, 14, 15]

## 3. Targeting Specific Ports

By default, Nmap only scans the 1,000 most common ports. You can change this using the -p flag: [7]

- Specific Ports: nmap -p 80,443 192.168.1.1
- Port Range: nmap -p 1-100 192.168.1.1
- All Ports: nmap -p- 192.168.1.1 (Scans all 65,535 possible ports) [7, 14, 16, 17]

## 4. Managing Speed & Output

- Timing Templates (-T0 to -T5): Control how aggressive the scan is. -T4 is recommended for stable networks, while -T1 is
  extremely slow and stealthy.
- Save Results:
- Normal Text: -oN filename.txt
  - XML (for other tools): -oX filename.xml
  - All Formats: -oA filename (Saves in normal, XML, and grepable formats at once) [10, 14, 18, 19, 20]

## Example "Power" Command

To perform a comprehensive scan on a target while saving the results:

sudo nmap -sS -sV -O -T4 -oA my_scan_results 192.168.1.1

This runs a stealthy SYN scan (-sS), detects services (-sV), identifies the OS (-O), uses fast timing (-T4), and saves all
output formats (-oA). [10, 11, 13, 21, 22]
