Network scanning is an investigative process used to identify active devices, open ports, and running services within a
computer network. Think of it as a digital "attendance check" that maps out what is connected to the network and how those
systems are communicating. [1, 2, 3]

## How Network Scanning Works

The process typically follows a sequential flow to gather information: [4]

- Host Discovery: The scanner identifies which devices (hosts) are currently online, often by sending "pings" (ICMP requests)
  to a range of IP addresses and waiting for a response.
- Port Scanning: For each active host, the scanner probes specific numerical "doors" called ports. This determines if they
  are open (accepting connections), closed, or filtered by a firewall.
- Service & OS Detection: The scanner analyzes responses to identify what software is running (e.g., a web server or
  database) and determines the device's operating system. [1, 2, 5, 6, 7, 8]

## Core Types of Scanning

- Active Scanning: The scanner proactively sends packets (probes) to devices and waits for them to talk back.
- Passive Scanning: Instead of sending probes, the scanner "listens" to existing network traffic to identify risks without
  alerting the target.
- Vulnerability Scanning: This goes a step further by comparing discovered software against databases of known security flaws
  to identify potential entry points for attackers. [1, 3, 5, 9, 10]

## Why It Is Used

Network scanning is a "double-edged sword" used by both defenders and attackers: [5, 8, 9, 11, 12]

- For Administrators: It is essential for managing assets, troubleshooting performance bottlenecks, ensuring security
  compliance (like HIPAA or PCI DSS), and finding "shadow IT" (unauthorized devices).
- For Ethical Hackers/Attackers: It serves as reconnaissance to find weak spots, outdated software, or misconfigured systems
  to exploit. [1, 10, 13, 14, 15, 16, 17]

## Popular Tools

- [Nmap](https://nmap.org/): The industry standard for network discovery and security auditing.
- [Angry IP Scanner](https://angryip.org/): A fast, cross-platform tool favored for its simplicity and GUI.
- [Wireshark](https://www.wireshark.org/): A network protocol analyzer often used for passive scanning and traffic
  inspection.
- [Nessus](https://www.tenable.com/products/nessus): A widely used tool specifically for deep vulnerability assessments. [9,
  16, 18, 19, 20, 21]
