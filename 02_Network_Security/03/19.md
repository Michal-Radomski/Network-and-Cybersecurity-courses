Hashcat is widely considered the world’s fastest and most advanced password recovery utility. It is an open-source tool used
by cybersecurity professionals and system administrators to "crack" or recover passwords from cryptographic hashes. [1, 2, 3,
4]

## What Makes It Unique?

The defining feature of Hashcat is its GPU acceleration. While traditional tools like John the Ripper often rely on the
computer's CPU, Hashcat is designed to harness the parallel processing power of graphics cards (NVIDIA, AMD, etc.). [4, 5, 6]

- Massive Speed: A high-end GPU can test billions or even trillions of password combinations per second—dramatically faster
  than a standard processor.
- Broad Support: It can crack over 450 different types of hashes, including those from Windows (NTLM), Linux (bcrypt), web
  applications (MD5, SHA), and encrypted files like ZIP or PDF. [1, 5, 6, 7, 8]

## Key Attack Modes

Hashcat offers five unique "attack modes" to efficiently guess passwords based on human habits: [2, 9]

- Dictionary Attack (-a 0): Tests words from a provided list of common passwords (like the famous rockyou.txt).
- Combinator Attack (-a 1): Combines words from two different lists (e.g., "Summer" + "2024").
- Brute-Force / Mask Attack (-a 3): Systematically tries every possible combination of characters within a defined pattern
  (e.g., "Capital letter + 6 lowercase letters + 2 numbers").
- Hybrid Attack (-a 6/7): Combines a dictionary word with a brute-force pattern, such as adding three random digits to the
  end of every word in a list.
- Rule-Based Attack: Applies complex "rules" to dictionary words, such as changing "e" to "3" or capitalizing the first
  letter. [5, 7, 10, 11, 12]

## Legitimate Use Cases

Ethical hackers and IT teams use [Hashcat](https://hashcat.net/hashcat/) for: [13]

- Security Auditing: Testing an organization’s password policy to see how many employee passwords can be cracked within a
  workday.
- Digital Forensics: Recovering passwords from evidence files in criminal investigations.
- Password Recovery: Helping users or administrators regain access to systems when a password has been lost. [4, 7, 13]

## Basic Example (Linux)

To run a simple dictionary attack on an MD5 hash using Hashcat on Ubuntu, you would use:

# -m 0 specifies MD5 hash type# -a 0 specifies dictionary attack# hash.txt is your target, rockyou.txt is your wordlist

hashcat -m 0 -a 0 hash.txt /usr/share/wordlists/rockyou.txt
