The deep web (or deep net) and the dark web (often called the dark net) are different layers of the internet that search
engines like Google cannot see. The simplest way to understand them is through the "iceberg" metaphor: [1, 2, 3, 4, 5]

## 1. The Deep Web (The invisible part of the iceberg)

The deep web refers to any part of the internet that isn't indexed by standard search engines. It accounts for roughly 90% to
95% of the entire internet. [4, 6, 7, 8]

- What it is: Routine, private content hidden behind logins, passwords, or paywalls.
- Examples: Your Gmail inbox, online banking portals, private cloud storage (like Google Drive), company intranets, and
  medical records.
- Accessibility: You access it every day using a standard web browser (Chrome, Safari, etc.) by logging in.
- Legality: It is entirely legal and essential for modern privacy and security. [8, 9, 10, 11]

## 2. The Dark Web (The hidden tip at the bottom)

The dark web is a very small, intentionally hidden subset of the deep web. It makes up less than 1% of the total internet.
[6, 10, 11, 12]

- What it is: Encrypted networks (called "darknets") that mask user identities and locations.
- Accessibility: It cannot be accessed with a normal browser; you need specialized software like the
  [Tor Browser](https://www.torproject.org/download/) (The Onion Router).
- Content: While famous for illegal marketplaces (selling drugs, stolen data, or weapons), it is also used for legitimate
  purposes like anonymous whistleblowing and helping activists in countries with heavy censorship.
- Legality: Simply accessing the dark web is legal in most countries, but engaging in illegal activities there is still a
  crime. [8, 10, 13, 14, 15, 16, 17]

## Key Differences at a Glance

| Feature [10, 12, 14, 18, 19] | Deep Web                          | Dark Web                         |
| ---------------------------- | --------------------------------- | -------------------------------- |
| Size                         | Massive (~90%+ of the web)        | Tiny (less than 1%)              |
| Access                       | Standard browser + login          | Specialized software (e.g., Tor) |
| Purpose                      | Privacy & restricted access       | Anonymity & hidden services      |
| Common Use                   | Banking, Email, Private databases | Whistleblowing, Illegal markets  |

Note on "Darknet": Technically, a darknet is the underlying infrastructure or private network itself (like Tor, I2P, or
Freenet), while the dark web refers to the websites and content hosted on those networks. [15, 17, 20]

To access the dark web on Ubuntu, you must use the Tor Browser, which is specifically designed to navigate .onion websites.
There are several ways to install it, but using Flatpak is currently recommended by many in the community because it is
officially maintained by the Tor Project and comes pre-configured with security profiles like AppArmor. [1, 2, 3, 4]

## Method 1: Using Flatpak (Recommended)

This method is considered safer because it handles dependencies and security settings automatically. [1, 4]

1.  Install Flatpak: Open your terminal ($Ctrl+Alt+T$) and run: sudo apt update && sudo apt install flatpak
2.  Add Flathub: flatpak remote-add --if-not-exists flathub https://flathub.org
3.  Install Tor Browser Launcher: flatpak install flathub org.torproject.torbrowser-launcher
4.  Launch: You can now find "Tor Browser" in your applications menu or run flatpak run org.torproject.torbrowser-launcher.
    [1, 5]

## Method 2: Direct Download from Tor Project

Use this if you prefer a "portable" version that doesn't require a full installation. [5, 6]

1.  Download: Go to the [official Tor Project download page](https://www.torproject.org/download/) and select the Linux
    package.
2.  Extract: Right-click the .tar.xz file and select "Extract Here," or use the command: tar -xf
    tor-browser-linux64-\*.tar.xz
3.  Run: Open the extracted folder, right-click start-tor-browser.desktop, select Properties > Permissions, and check "Allow
    executing file as program". Double-click it to start. [6, 7, 8, 9, 10, 11]

## Critical Safety Tips for Your First Visit

The dark web has no oversight and can be dangerous. [12, 13]

- Security Level: Once Tor opens, click the "Shield" icon next to the URL bar and set the security level to "Safer" or
  "Safest" to disable risky scripts like JavaScript.
- Use a VPN: Connecting to a [VPN](https://www.bleepingcomputer.com/vpn/guides/access-dark-web-safely/) before opening Tor
  hides your Tor usage from your Internet Service Provider (ISP).
- Don't Share Info: Never use your real name, email, or any personal details on .onion sites.
- Avoid Downloads: Do not download files from the dark web, as they often contain malware. [2, 14, 15, 16, 17, 18]
