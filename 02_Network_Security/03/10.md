Spytech SpyAgent is a comprehensive computer monitoring and "spy" software designed to track and record nearly every activity
performed on a Windows computer. [1, 2, 3] Developed by Spytech Software, Inc., it has been on the market for over 25 years
and is primarily marketed to parents for monitoring children and to business owners for tracking employee productivity. [1,
4]

## Key Features

- Activity Logging: It records keystrokes (including passwords), websites visited, emails sent/received, chat conversations,
  applications used, and file transfers.
- Visual Monitoring: The software captures screenshots of the desktop and can even take images via the webcam or record audio
  through the microphone.
- Stealth Mode: It can run in a completely invisible fashion, hiding from the Task Manager and typical system menus to
  prevent the user from knowing they are being monitored.
- Content Filtering: Beyond monitoring, it can block specific websites, applications, and chat clients based on keywords or
  categories.
- Remote Access: When paired with their cloud service (SpyAnywhere), logs can be viewed remotely via a web browser in
  real-time. [1, 2, 5, 6, 7, 8, 9]

## Usage and Ethics

While marketed as a security and monitoring tool for legal use by parents and employers, it is categorized by security labs
like [FortiGuard](https://www.fortiguard.com/appcontrol/17740) as a tool capable of unexpected network communication and
behavior similar to a botnet because of its stealth and data-exfiltration capabilities. [10, 11]

In the context of Spytech SpyAgent, Stealth Mode is a specialized operating state that allows the software to run with total
invisibility to the user being monitored. When active, it ensures the software does not appear in common system areas where
it might be detected. [1, 2]

## Hidden Elements

While in Stealth Mode, SpyAgent is removed from the following locations:

- Task Manager: It does not appear in the active process or application lists.
- Desktop and Taskbar: There are no icons, windows, or notifications visible to the user.
- Start Menu and Add/Remove Programs: The software can hide its entry in the Windows Start menu and the Control Panel's
  uninstaller list.
- System Folders: It can even hide its own installation and log folders to prevent them from being found during manual file
  searches. [1, 2, 3, 4]

## Accessing the Software

Because it is invisible, the administrator must use specific "secret" methods to bring the software out of stealth: [5]

1.  Hotkey Combination: By default, pressing CONTROL + ALT + SHIFT + M simultaneously triggers a password prompt.
2.  Key Phrase Access: Users can configure a specific word or phrase that, when typed anywhere, brings up the interface.
3.  NoStealth Utility: A special file named nostealth.exe located in the installation directory can be run to manually reveal
    the program. [1, 4, 6, 7, 8]

## Stealth Edition

Spytech also offers a Stealth Edition of SpyAgent, which is a pre-configured version that installs and begins monitoring in
total stealth automatically, requiring no manual configuration of these settings by the person installing it. [9, 10, 11]
