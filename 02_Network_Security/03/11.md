A keylogger (short for "keystroke logger") is a tool designed to record every key pressed on a computer or mobile device.
While they can be used for legitimate monitoring—such as parents tracking their children or employers overseeing company
devices—they are most commonly associated with malware used to steal sensitive data like passwords and credit card numbers.
[1, 2, 3, 4]

## Main Types of Keyloggers

Keyloggers fall into two primary categories based on how they are installed and operate: [5, 6, 7]

- Software-Based Keyloggers: These are the most common and are typically installed remotely via phishing emails, malicious
  downloads, or "drive-by" infections from compromised websites.
- API-Based: Hooks into the operating system's functions to record inputs before they reach an app.
  - Kernel-Level: Operates deep within the OS core, making them extremely difficult for standard antivirus software to
    detect.
  - Form-Grabbing: Specifically targets web browsers to steal data (like login credentials) as it is typed into web forms,
    before it is even encrypted.
- Hardware-Based Keyloggers: These are physical devices that must be manually connected to a computer.
- Keyboard Adapters: Tiny dongles plugged between the keyboard cable and the computer's USB port.
  - Built-in Hardware: Modified keyboards or internal components with pre-installed logging chips. [1, 2, 3, 8, 9, 10, 11,
    12, 13]

## How They Spread and Work

Most malicious keyloggers arrive through standard malware delivery methods: [14]

- Phishing: Disguised as legitimate attachments or links in emails.
- Trojan Horses: Hidden inside free or pirated software downloads.
- Web Scripts: Malicious code on a website that automatically downloads to your device upon visiting. [3, 10, 12, 15, 16, 17]

Once active, the logger creates a "transcript" of your typing and periodically transmits this file to a remote server
controlled by the attacker. [1, 14]

## Warning Signs of an Infection

Because keyloggers are designed to be stealthy, they often don't cause obvious system errors. However, you might notice: [3,
18]

- System Lag: Noticeable delays between typing a key and seeing it appear on screen.
- Performance Drops: Unusual slowdowns, high CPU usage, or rapid battery drain on mobile devices.
- Strange Network Activity: Spikes in data usage as the logger sends logs to an external server. [19, 20, 21, 22, 23]

## How to Protect Yourself

1.  Use Multi-Factor Authentication (MFA): Even if a keylogger steals your password, the attacker won't have the secondary
    code needed to log in.
2.  Use a Password Manager: These tools use "autofill" to enter passwords, which bypasses the keyboard entirely and leaves
    nothing for a keylogger to record.
3.  Regular Scans: Use trusted antivirus software to detect and remove software-based threats.
4.  Physical Inspection: Periodically check the back of your computer for unfamiliar USB devices. [1, 3, 10, 18, 19, 24]
