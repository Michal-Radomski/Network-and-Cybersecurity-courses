JPS Virus Maker is a legacy, Windows-based graphical tool used to create simple, custom malware and "prank" viruses. It is
primarily used today as an educational tool in cybersecurity labs to teach students how malicious payloads are structured and
how antivirus systems detect them. [1, 2, 3, 4]

## Key Features and Capabilities

The tool allows users to select various harmful or annoying actions by checking boxes in a GUI: [3, 5, 6, 7]

- System Disruption: Forcing restarts, shutting down the computer, or terminating Windows processes.
- Security Disabling: Turning off the Windows Security Center, firewalls, and common antivirus programs like Norton or
  McAfee.
- User Harassment: Locking the mouse and keyboard, hiding the taskbar or system clock, and opening endless browser windows.
- Persistence: Features like Auto Startup ensure the virus runs every time the computer boots.
- Redirection: It can redirect the victim's browser to a specific website; the default for version 4.0 is often
  http://www.jpsvirus.net. [2, 3, 6, 8, 9, 10]

## Usage and Effectiveness

- Ethical Hacking Labs: It is a standard component of certification courses like the
  [Certified Ethical Hacker (CEH)](https://blog.rootkid.in/exam-prep-notes/certified-ethical-hacker-v12-practical-notes/module-07-malware-threats)
  to demonstrate proof-of-concept attacks in sandboxed environments.
- Modern Compatibility: While very effective on older versions of Windows (XP, Vista, 7), it is largely ineffective against
  modern security features in Windows 10 and 11.
- Detection: Because the tool uses static, well-known code templates, most modern antivirus programs detect viruses generated
  by JPS almost instantly. [8, 11, 12, 13, 14]

Warning: Using JPS Virus Maker on any system without explicit authorization is illegal and unethical. It should only be used
in isolated, virtualized lab environments for learning purposes. [2, 6, 15]
