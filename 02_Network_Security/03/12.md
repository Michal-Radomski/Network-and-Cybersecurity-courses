LOIC (Low Orbit Ion Cannon) and HOIC (High Orbit Ion Cannon) are open-source network stress-testing tools frequently used to
launch Denial-of-Service (DoS) and Distributed Denial-of-Service (DDoS) attacks. Both are named after fictional weapons from
the Command & Conquer video game series. [1, 2, 3]

## What is LOIC?

LOIC is a simple, user-friendly tool that floods a target server with TCP, UDP, or HTTP packets. It was originally developed
by Praetox Technologies for legitimate network testing but became famous when adopted by the hacktivist group Anonymous for
coordinated attacks. [2, 4]

- Coordination: Often uses a "Hivemind" mode via IRC, allowing a single person to control many voluntary users' computers at
  once.
- Traceability: It does not mask the attacker's IP address, making participants easy to identify and prosecute. [4]

## What is HOIC?

HOIC was developed as a more advanced successor to LOIC to fix its predecessor's weaknesses, specifically regarding detection
and "firepower". It focuses primarily on HTTP flood attacks at the application layer. [3, 5, 6, 7]

## Key Differences

The primary evolution from LOIC to HOIC involves the efficiency and stealth of the attack. [8]

| Feature [1, 3, 4, 5, 6, 7, 9, 10] | LOIC (Low Orbit)                                     | HOIC (High Orbit)                                                |
| --------------------------------- | ---------------------------------------------------- | ---------------------------------------------------------------- |
| Attack Vector                     | TCP, UDP, and HTTP floods.                           | Primarily HTTP GET and POST floods.                              |
| Efficiency                        | Requires thousands of users for an effective attack. | Can be effective with as few as 50 users.                        |
| Anonymity                         | None; user IPs are fully visible to the target.      | Uses Booster Scripts to randomize headers and obfuscate traffic. |
| Targets                           | Typically floods a single IP/target at a time.       | Can target up to 256 URLs simultaneously.                        |
| Customization                     | Manual configuration only.                           | Supports customizable scripts to vary attack patterns.           |

For technical defensive measures, organizations often use services like
[Cloudflare](https://www.cloudflare.com/learning/ddos/ddos-attack-tools/high-orbit-ion-cannon-hoic/) or
[Imperva](https://www.imperva.com/learn/ddos/high-orbit-ion-cannon/) to mitigate the volumetric traffic these tools generate.
