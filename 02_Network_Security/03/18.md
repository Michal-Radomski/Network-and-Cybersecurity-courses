A rainbow table is a large, precomputed database used to reverse-engineer password hashes into their original plaintext form.
[1, 2] Instead of guessing passwords in real-time (like a brute-force attack), an attacker uses these tables to look up a
stolen hash. If a match is found, the original password is revealed almost instantly. [1, 3]

## How They Work

- Time-Memory Trade-off: Storing every possible password and its hash would require massive amounts of storage. Rainbow
  tables use "chains" and reduction functions to compress this data, storing only the start and end points of a chain to save
  space while keeping lookup speeds fast.
- Hash-Specific: Each table is built for a specific algorithm (e.g., MD5, SHA-1, or NTLM) and a specific character set (e.g.,
  all lowercase letters, 1-8 characters long).
- Ineffectiveness Against Salts: Modern systems use salting—adding random data to a password before hashing it. This makes
  precomputed rainbow tables useless because an attacker would need a unique table for every possible salt. [1, 4, 5, 6, 7,
  8]

## How to Use Rainbow Tables on Linux

Linux users, particularly those on security-focused distributions like [Kali Linux](https://www.kali.org/), typically use
RainbowCrack or Ophcrack to perform these attacks. [9]

## 1. Using RainbowCrack

RainbowCrack is a suite of tools used to generate, sort, and search rainbow tables. [10, 11]

- Generate a table: Use rtgen to create a new table for a specific hash type.

./rtgen md5 loweralpha-numeric 1 5 0 3800 33554432 0

- Sort the table: Before searching, tables must be sorted using rtsort to optimize performance.

./rtsort .

- Crack the hash: Use rcrack to search your sorted tables for a specific hash.

./rcrack . -h [stolen_hash_here]

[12, 13, 14]

## 2. Using Ophcrack

Ophcrack is an easy-to-use tool with a graphical interface (GUI) specifically designed for cracking Windows (LM/NTLM)
passwords. [9, 15]

1.  Install: Open a terminal and run sudo apt install ophcrack.
2.  Load Tables: You must download pre-made tables from the Ophcrack website first.
3.  Run: Click Tables -> Install to point the program to your downloaded tables.
4.  Load Hashes: Use Load -> Single hash (or load a file containing multiple hashes) and click Crack. [9, 15, 16]

Note: For more modern and complex passwords (like those on Ubuntu or modern Windows versions), it is usually faster to use
Hashcat with GPU acceleration rather than relying on massive rainbow tables. [10]
