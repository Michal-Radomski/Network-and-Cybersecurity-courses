To configure Snort and Suricata on Ubuntu, the process generally involves updating system repositories, installing the
software via a package manager or PPA, and then manually editing their configuration files to define your network and
rulesets. [1]

## 1. How to Configure Snort

Snort is often installed using the standard apt package manager for a quick setup. [2]

- Installation:

1. Update your system: sudo apt update && sudo apt upgrade -y. 2. Install Snort: sudo apt install snort -y. 3. During
   installation, you will be prompted to enter your Address Range (e.g., 192.168.1.0/24) and the Network Interface to monitor
   (e.g., eth0 or enp0s3).

- Configuration:

1. Edit Config: The main file is at /etc/snort/snort.conf. Use sudo nano /etc/snort/snort.conf to adjust variables like ipvar
   HOME_NET. 2. Validate: Test your configuration file for errors before running: sudo snort -T -c /etc/snort/snort.conf. 3.
   Run Snort: Start it in sniffer mode to see live alerts: sudo snort -A console -q -u snort -g snort -c
   /etc/snort/snort.conf -i eth0. [1, 2, 3, 4, 5, 6, 7]

---

## 2. How to Configure Suricata

Suricata is best installed via the official
[OISF PPA](https://www.digitalocean.com/community/questions/how-to-install-and-configure-suricata-on-ubuntu-22-04) to ensure
you have the latest stable version. [8, 9]

- Installation:

1. Add the PPA: sudo add-apt-repository ppa:oisf/suricata-stable. 2. Update and install: sudo apt update && sudo apt install
   suricata -y.

- Configuration:

1. Define Network: Open the configuration at /etc/suricata/suricata.yaml and set your HOME_NET to match your local subnet. 2.
   Set Interface: Under the af-packet section, ensure the interface name matches your active network card (e.g., eth0). 3.
   Update Rules: Suricata uses a built-in tool to fetch the latest "Emerging Threats" rules: sudo suricata-update. 4. Test &
   Start:
   - Validate config: sudo suricata -T -c /etc/suricata/suricata.yaml -v.
     - Start the service: sudo systemctl start suricata. [8, 9, 10, 11, 12, 13, 14, 15, 16]

## Key Comparison for Ubuntu

| Feature [2, 9, 11, 13, 17] | Snort (v2 Default)   | Suricata                               |
| -------------------------- | -------------------- | -------------------------------------- |
| Config Format              | .conf (Text-based)   | .yaml (Structured)                     |
| Threading                  | Single-threaded      | Multi-threaded (Faster on modern CPUs) |
| PPA Available              | No (Standard repos)  | Yes (Official OISF PPA)                |
| Rule Update                | Manual or PulledPork | suricata-update (Built-in)             |
