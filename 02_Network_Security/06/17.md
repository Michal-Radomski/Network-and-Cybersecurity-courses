Wazuh is a free, open-source security platform that combines SIEM (Security Information and Event Management) and XDR
(Extended Detection and Response). [1, 2] While a firewall is a "gatekeeper" that blocks traffic at the entrance, Wazuh is a
"central command center" that watches everything happening inside your systems. [3, 4, 5]

## Key Differences: Wazuh vs. Firewall

| Feature [6, 7, 8, 9, 10, 11, 12, 13] | Firewall (Network/Host)                                | Wazuh (SIEM/XDR)                                                                                                  |
| ------------------------------------ | ------------------------------------------------------ | ----------------------------------------------------------------------------------------------------------------- |
| Primary Goal                         | Prevention: Blocks unauthorized access based on rules. | Detection & Response: Monitors logs and behavior to find threats.                                                 |
| Visibility                           | Sees IP addresses, ports, and data packets.            | Sees file changes, user logins, running processes, and vulnerabilities.                                           |
| Action                               | Drops or rejects "bad" traffic instantly.              | Analyzes data to alert you of a breach and can trigger "Active Responses" (like running a script to block an IP). |
| Focus                                | Network perimeter or OS traffic boundaries.            | Comprehensive endpoint and cloud workload protection.                                                             |

## What Wazuh Does (That a Firewall Doesn't)

- File Integrity Monitoring (FIM): Alerts you immediately if a critical system file is modified or deleted.
- Vulnerability Detection: Automatically scans your installed software for known security holes (CVEs) and tells you what to
  patch.
- Log Correlation: It collects logs from your firewalls, servers, and apps into one "single pane of glass" to find patterns
  that a single device would miss.
- Compliance Auditing: Checks your system settings against security standards like PCI DSS, HIPAA, or GDPR. [1, 7, 9, 14, 15,
  16]

## How They Work Together

They are not competitors; they are partners. You can actually integrate your firewall logs into Wazuh. If your firewall
detects an attack, it sends a log to Wazuh; Wazuh then analyzes that attack alongside your server's behavior to decide if a
broader security response is needed. [3, 15, 17, 18, 19]
