A Web Application Firewall (WAF) is a specialized security tool that acts as a "shield" between a web application and the
internet. Unlike a standard firewall that guards an entire network, a WAF specifically protects Layer 7 (the Application
Layer) of the OSI model. [1, 2, 3, 4] It functions as a reverse proxy, inspecting every incoming HTTP/HTTPS request before it
reaches the web server to block malicious activity that traditional firewalls often miss. [1, 5]

---

## Core Protections

WAFs are primarily designed to defend against the OWASP Top 10, a list of the most critical web application security risks.
[4, 6]

- SQL Injection (SQLi): Prevents attackers from injecting malicious database queries to steal or delete data.
- Cross-Site Scripting (XSS): Blocks malicious scripts from being executed in a user's browser.
- Cookie Poisoning: Stops unauthorized modification of cookies to bypass security.
- Virtual Patching: Blocks exploits for known vulnerabilities instantly, giving developers time to fix the actual code. [1,
  3, 4, 7, 8, 9]

---

## How It Differs from Other Firewalls

While they share the same name, they serve very different purposes: [10, 11]

| Feature [3, 12, 13, 14, 15] | Network Firewall (NGFW)            | Web Application Firewall (WAF)        |
| --------------------------- | ---------------------------------- | ------------------------------------- |
| OSI Layer                   | Layers 3 & 4 (Network/Transport)   | Layer 7 (Application)                 |
| Primary Goal                | Blocks unauthorized network access | Blocks application-level attacks      |
| Visibility                  | Port & IP addresses                | HTTP methods, headers, and payloads   |
| Focus                       | Protecting the user and network    | Protecting the server and application |

---

## 🛡️ Implementation Types

- Cloud-Based: Offered as a SaaS (Software-as-a-Service). It is the easiest to set up (often via a DNS change) and receives
  automatic updates from the provider.
- Host-Based: Integrated directly into the application server's software. It is highly customizable but consumes the server's
  local resources.
- Network-Based: A physical hardware appliance installed locally. It offers the lowest latency but is the most expensive to
  buy and maintain. [1, 9, 16, 17, 18]

---

## Why Use a WAF?

Organizations use WAFs because they provide behavioral analytics to identify unusual traffic spikes, such as bot attacks or
Layer 7 DDoS attempts, ensuring that a website remains available even during an attack. Additionally, they are often a
requirement for compliance standards like PCI DSS for companies handling credit card information. [2, 3, 5, 8]
