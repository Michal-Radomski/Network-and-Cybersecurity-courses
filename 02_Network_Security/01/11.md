While Ethernet is the global standard for wired networking today, Token Ring and Frame Relay are legacy technologies that
were once its primary competitors in local and wide area networks. [1, 2, 3, 4]

## Ethernet

Ethernet is the most widely used technology for Local Area Networks (LANs). It is standardized as IEEE 802.3. [5, 6, 7]

- How it Works: It uses a "contention-based" method called CSMA/CD (Carrier Sense Multiple Access with Collision Detection).
  Devices "listen" to the cable; if it's quiet, they send data. If two devices speak at once (a collision), they both stop,
  wait a random time, and try again.
- Status: Dominant. It won out because it is inexpensive, easy to install, and has evolved from 10 Mbps to over 400 Gbps. [6,
  7, 8, 9]

## Token Ring

Developed by IBM in the 1980s, Token Ring (IEEE 802.5) was Ethernet’s main rival for decades. [10, 11]

- How it Works: Devices are connected in a logical circle. A small data packet called a token circulates the ring. A device
  can only transmit data if it "captures" the token. This prevents data collisions entirely, making it very efficient for
  heavy traffic.
- Status: Obsolete. Although it was more "orderly" than early Ethernet, the hardware was significantly more expensive and
  complex to maintain. [1, 9, 12, 13, 14]

## Frame Relay

Unlike the first two, Frame Relay is a Wide Area Network (WAN) technology used to connect geographically distant LANs. [15,
16]

- How it Works: It is a "packet-switching" protocol that sends data in variable-sized frames over a shared network. It uses
  Virtual Circuits (VCs) to create logical paths between routers, allowing multiple customers to share the same physical
  fiber lines while acting as if they have a private connection.
- Status: Largely Legacy. Once the standard for business internet in the 90s, it has been replaced by more modern
  technologies like MPLS and high-speed Metro Ethernet. [2, 16, 17, 18]
