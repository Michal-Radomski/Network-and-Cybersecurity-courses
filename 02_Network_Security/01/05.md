Network components are the essential hardware and software elements used to build, manage, and secure a computer network.
They can be categorized into four primary groups: [1, 2, 3, 4, 5]

## 1. Intermediary Networking Devices [6]

These devices manage and direct traffic between various parts of a network or between different networks. [6, 7, 8]

- [Routers](https://www.lepide.com/blog/the-most-common-types-of-network-devices/): Connect different networks (e.g., your
  home network to the internet) and route data packets based on IP addresses.
- [Switches](https://www.cbtnuggets.com/blog/technology/networking/the-3-components-of-a-network-explained): Connect multiple
  devices within a single Local Area Network (LAN) and use MAC addresses to send data to the correct recipient.
- [Modems](https://netwrix.com/en/resources/blog/network-devices-explained/): Convert digital data into signals suitable for
  transmission over analog lines (like cable or phone lines) and vice versa.
- Wireless Access Points (WAPs): Allow wireless devices to connect to a wired network using Wi-Fi.
- Firewalls: Security devices that monitor and filter incoming and outgoing network traffic based on predefined rules.
- Hubs and [Repeaters](https://www.purplerockdubai.com/understanding-the-eight-essential-components-of-a-network/): Older or
  simpler devices used to connect multiple devices or extend signal range by amplifying it. [2, 3, 6, 7, 8, 9, 10, 11]

## 2. End Devices (Nodes)

These are the source or destination points for data in a network. [9, 12]

- Computers & Laptops: Standard user devices for data access and creation.
- [Servers](https://en.wikiversity.org/wiki/Basic_computer_network_components): Powerful computers that host data,
  applications, or services (like email or web hosting) for other devices on the network.
- Printers & Peripherals: Shared hardware resources such as network-connected printers.
- IoT Devices: Smart cameras, sensors, and other "connected" objects. [1, 3, 9, 12, 13]

## 3. Connection Components

These are the physical or wireless means used to link devices together. [1, 12, 14]

- [Network Interface Cards (NICs)](https://www.geeksforgeeks.org/computer-networks/physical-components-of-computer-network/):
  Hardware installed in a device that allows it to physically connect to the network media.
- Transmission Media:
- Wired: Ethernet (twisted-pair), Fiber-optic (fast, long-distance), and Coaxial cables.
  - Wireless: Radio waves (Wi-Fi, Bluetooth), microwaves, or infrared. [1, 6, 9, 12, 14]

## 4. Software and Logical Components

The intangible rules and programs that allow hardware to communicate. [3, 12, 15]

- Protocols: Set of standardized rules (like TCP/IP, HTTP, or DNS) that define how data is formatted and transmitted.
- Network Operating System (NOS): Software that manages network resources and security (e.g., Windows Server, Linux). [9, 12,
  13, 15, 16]
