tcpdump is a powerful, command-line packet analyzer. It allows you to intercept and display TCP/IP and other packets being
transmitted or received over a network. [1, 2, 3, 4, 5] Think of it as the "Swiss Army Knife" of network troubleshooting.
While Wireshark has a GUI and fancy graphs, tcpdump is lightweight, runs in the terminal, and is perfect for remote servers
where no desktop interface is available. [6, 7, 8, 9, 10]

## How to Install it

It usually comes pre-installed on Ubuntu, but if not: sudo apt update && sudo apt install tcpdump -y [11]

---

## Common tcpdump Commands

To use tcpdump, you generally need sudo because it puts the network card into promiscuous mode. [12, 13, 14]

## 1. List available interfaces

Before sniffing, see which network cards (Ethernet, Wi-Fi) you can listen to: sudo tcpdump -D [15]

## 2. Capture all traffic on a specific interface [16, 17]

Replace eth0 or wlan0 with your interface name: sudo tcpdump -i eth0 [18, 19, 20]

## 3. Capture and show only 10 packets (then stop) [21]

sudo tcpdump -c 10 -i eth0 [22]

## 4. Filter by Port (e.g., only Web traffic) [23]

sudo tcpdump -i eth0 port 80 [24]

## 5. Filter by Source IP

To see only packets coming from a specific machine: sudo tcpdump -i eth0 src 192.168.1.5

## 6. Save a capture to a file (for Wireshark) [25, 26]

If you want to record the traffic now and analyze it later in a GUI like Wireshark: sudo tcpdump -i eth0 -w output_file.pcap
[27, 28, 29, 30]

---

## Pro-Tips for Reading Output

- -n: Don't resolve hostnames (shows IP addresses instead of names). This makes the tool much faster.
- -A: Print the packet contents in ASCII. This allows you to read the actual text of unencrypted messages (like HTTP or
  Telnet).
- -v: Verbose output (provides more technical detail like TTL and ID numbers). [31, 32, 33, 34]

A word of caution: If you run tcpdump on a high-traffic server without filters, your terminal will be flooded with data
instantly. Always try to use a filter (like port 22) to narrow down what you are looking for. [35, 36]
