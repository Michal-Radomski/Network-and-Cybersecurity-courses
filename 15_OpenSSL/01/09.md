Since Secure Boot is switched off, your computer is likely bypassing Shim and launching GRUB directly, but the Shim files are
almost certainly still sitting in your EFI partition.

Because Windows 10 and Ubuntu 24.04 share the same EFI System Partition (ESP) on a dual-boot machine, you can check for Shim
from either operating system. Here is how to do it.

---

## Method 1: Check from Ubuntu (Easiest)

On Ubuntu, you can look directly inside your boot directory or use the package manager to see if the Shim package is
installed.

### Step 1: Look for the Shim binary file

Open your terminal (`Ctrl + Alt + T`) and list the contents of your Ubuntu EFI directory by running:

```bash
ls /boot/efi/EFI/ubuntu/

```

**What to look for:** Look for a file named **`shimx64.efi`**.

- If you see `shimx64.efi`, Shim is installed on your drive.
- If you only see `grubx64.efi`, your system is relying solely on GRUB.

### Step 2: Check the Package Manager

To verify if Ubuntu is actively maintaining the Shim software, run:

```bash
dpkg -l | grep shim

```

If it returns lines showing `shim` and `shim-signed`, the software is installed on your Linux system.

---

## Method 2: Check from Windows 10

Because the EFI partition is hidden in Windows by default, you have to use the command prompt as an Administrator to peek
inside it.

### Step 1: Open Command Prompt as Admin

1. Click the Start menu, type **cmd**.
2. Right-click **Command Prompt** and select **Run as administrator**.

### Step 2: Mount the Hidden EFI Partition

Run the following command to assign a temporary drive letter (like `Z:`) to your hidden EFI partition:

```cmd
mountvol Z: /S

```

### Step 3: Navigate and Check the Directory

Now, change to the `Z:` drive and look inside the Ubuntu folder:

```cmd
Z:
cd EFI\ubuntu
dir

```

**What to look for:** Just like in Linux, look through the list of files for **`shimx64.efi`**.

### Step 4: Clean Up

Once you are done, close the command prompt. The `Z:` drive will automatically disappear when you restart your computer,
keeping the partition safe.

---

## What does this mean for you?

If you found `shimx64.efi`, it means your laptop is completely ready for Secure Boot. If you ever decide to go into your
BIOS/UEFI settings and turn Secure Boot back on, Ubuntu 24.04 will seamlessly boot through Shim without any extra
configuration.
