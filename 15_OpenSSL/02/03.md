Generating DSA (Digital Signature Algorithm) keys using OpenSSL involves a two-step process: first, you generate the DSA
parameters, and then you use those parameters to generate the actual key pair (private and public keys).

> ⚠️ **Note:** Modern cryptographic standards generally favor **RSA** or **Ed25519** (ECC) over DSA for security and
> performance reasons. However, if you need DSA for a specific legacy system or assignment, here is how you do it.

---

### Step 1: Generate DSA Parameters

DSA requires a set of pre-computed parameters (primes $p$, $q$, and generator $g$) before a key can be created. Run the
following command to generate a parameter file:

```bash
openssl dsaparam -out dsa_param.pem 2048

```

- `dsaparam`: Tells OpenSSL to use the DSA parameter generation tool.
- `-out dsa_param.pem`: Specifies the name of the file to save the parameters to.
- `2048`: The key size in bits (2048-bit is the standard minimum for DSA security).

---

### Step 2: Generate the Private Key

Once you have the parameter file, you can generate your actual DSA private key:

```bash
openssl gendsa -out dsa_priv.pem dsa_param.pem

```

- `gendsa`: Tells OpenSSL to generate a DSA key.
- `-out dsa_priv.pem`: The output file that will contain your new **private key**.
- `dsa_param.pem`: The parameter file you created in Step 1.

_(Optional)_ If you want to protect your private key with a password, you can add a cipher flag like `-aes256`:

```bash
openssl gendsa -aes256 -out dsa_priv.pem dsa_param.pem

```

---

### Step 3: Extract the Public Key

Your private key file actually contains both the private key and the public key components. To extract just the **public
key** so you can share it with others, use this command:

```bash
openssl dsa -in dsa_priv.pem -pubout -out dsa_pub.pem

```

- `dsa`: The DSA key processing utility.
- `-in dsa_priv.pem`: Specifies your input private key file.
- `-pubout`: Instructs OpenSSL to output the public key.
- `-out dsa_pub.pem`: The file where your public key will be saved.

---

### Summary of Files Created

- `dsa_param.pem`: Shared parameters (can be deleted after key generation).
- `dsa_priv.pem`: **Keep this secret!** This is your private key used for signing.
- `dsa_pub.pem`: Share this freely. This is your public key used for verification.
