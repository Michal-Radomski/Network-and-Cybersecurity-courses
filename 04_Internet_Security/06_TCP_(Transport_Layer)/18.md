In the world of Linux, the exclamation mark `!` is a **history expansion** character. It tells the shell (like Bash or Zsh)
to look back at your previously executed commands and run one of them again.

The command `!sudo` specifically tells your terminal to:

> **Find the most recent command in your history that starts with the string "sudo" and execute it immediately.**

---

### How it Works

When you type `!sudo`, the shell scans your command history from the most recent entry backward.

- If your last command was `sudo apt update`, then typing `!sudo` will run `sudo apt update` again.
- If you ran `ls`, then `mkdir test`, then `sudo su`, typing `!sudo` will skip the first two and run `sudo su`.

### Common Related Commands

History expansion is a powerful "lazy" tool. Here are a few cousins of `!sudo`:

| Command     | Action                                                                                                                       |
| ----------- | ---------------------------------------------------------------------------------------------------------------------------- |
| `!!`        | **The "Double Bang":** Runs the exact previous command. (Commonly used as `sudo !!` when you forget to use root privileges). |
| `!$`        | **Last Argument:** Grabs the last word of the previous command.                                                              |
| `!n`        | Runs the $n^{th}$ command from your history list.                                                                            |
| `!?string?` | Runs the most recent command that _contains_ "string" anywhere in the line.                                                  |

---

### ⚠️ A Small Warning

Using `!` commands can be a bit like "coding with your eyes closed." If you aren't 100% sure what your last `sudo` command
was, you might accidentally run something destructive (like a recursive delete or a system change) that you didn't intend to
repeat.

**Pro Tip:** If you want to see what the command is _before_ it runs, you can type `:p` at the end (e.g., `!sudo:p`). This
prints the command to the screen without executing it, adding it to your history so you can just hit "Up" and "Enter" to run
it safely.
