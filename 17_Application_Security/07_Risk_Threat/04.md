In application security, **Deciduous** isn't a standard security term or broad technical concept—it's actually the name of a
specific **open-source security tool**.

Here is what it is and how tree concepts work in app sec:

---

## 1. Deciduous (The Security Tool)

**[Deciduous](https://www.deciduous.app/)** is a threat-modeling app (and VS Code extension) designed by security experts
Kelly Shortridge and Ryan Petrich. It simplifies building **decision trees** and **attack trees** to model potential security
incidents.

- **What it does:** It lets developers and security teams map out system facts, potential attacker moves (attacks), and
  defensive countermeasures (mitigations) in simple code (YAML).
- **The Goal:** It visualizes how an attack might unfold step-by-step, helping teams spot weak points and decide where to add
  security controls before something goes wrong.

---

## 2. Attack Trees vs. Decision Trees (The Security Concept)

If someone mentioned a "tree" in app security, they are usually talking about **Attack Trees** or **Decision Trees**:

| Concept         | What It Means in App Security                                                             | Example                       |
| --------------- | ----------------------------------------------------------------------------------------- | ----------------------------- |
| **Attack Tree** | A diagram showing the different paths an attacker could take to compromise a system goal. | **Goal:** Steal Database <br> |

<br> ↳ **Path A:** SQL Injection <br>

<br> ↳ **Path B:** Stolen Admin Credentials | | **Decision Tree** | A diagram mapping out attacker choices alongside defender
reactions to model resilience. | **Attacker:** Uploads malicious file <br>

<br> ↳ **Defender:** Antivirus blocks file <br>

<br> ↳ **Attacker:** Uses obfuscated script |

---

> **Summary:** While a "deciduous tree" in biology is just a tree that sheds its leaves, in **application security, Deciduous
> is an app used for threat modeling**, built around security decision trees.
