Physical security in cybersecurity means protecting the _real‑world_ components that support digital systems—such as servers,
network devices, data centers, and user workstations—from theft, tampering, or unauthorized physical access.
[cyberpedia.reasonlabs](https://cyberpedia.reasonlabs.com/EN/physical%20security.html)

### What physical security covers

Physical security protects hardware, facilities, and people so that attackers cannot walk up to a server, plug in a rogue
device, or steal a laptop and then launch a cyberattack or exfiltrate data. It includes measures like locked doors, badges or
biometric access, surveillance cameras, and guards around sensitive areas (for example, data centers or server rooms).
[pavion](https://pavion.com/resource/cybersecurity-and-physical-security-a-combined-approach/)

### Why it matters for cybersecurity

Even the strongest firewalls and encryption are useless if an attacker can physically access a machine or network port. An
intruder with a USB drive, a smartphone, or a small device can plug into a network, install malware, or clone drives, turning
a “physical” event into a serious cyber incident.
[buildings.honeywell](https://buildings.honeywell.com/us/en/brands/our-brands/lenels2/news/insights/physical-and-cybersecurity)

### Common physical‑security controls

Typical controls used in a cybersecurity context include:

- Access control systems (keycards, biometrics, PIN‑pads) to restrict who enters sensitive areas.
  [coursera](https://www.coursera.org/articles/physical-security)
- Surveillance (CCTV, motion sensors) to detect and log suspicious activity around infrastructure.
  [securithings](https://securithings.com/physical-security-software/physical-security-cybersecurity/)
- Physical barriers such as fences, locked server racks, and secured cabinets to make it harder to reach critical hardware.
  [garda](https://www.garda.com/articles/what-is-physical-security-and-how-does-it-work)

### How physical and cyber security work together

Modern security frameworks treat physical and cyber security as fully connected: physical systems (cameras, access‑control
panels, IoT devices) are themselves part of the network and must be protected from cyberattack. In practice, organizations
coordinate both teams so that cyber alerts can trigger physical responses (like locking doors) and physical breaches can
trigger cyber‑incident workflows.
