**Traceroute** is a network diagnostic tool used to **track the path (route) that data takes from your device to a
destination server**.

### ⚡ Key points

- Shows each **hop** (router) along the way
- Measures **latency** (time) between hops
- Helps identify where network delays or failures occur

### 🧩 How it works

Traceroute sends packets with increasing **TTL (Time To Live - TTL defines the maximum number of "hops" (routers))** values:

1. First packet stops at the first router
2. Second packet reaches the next router
3. Continues until it reaches the destination

Each router sends back a response (usually using **ICMP**), revealing its position in the path.

### 🧠 Simple example

Running `traceroute google.com` might show:

- Your router → ISP → regional network → destination server

👉 In short: **Traceroute maps the journey your data takes across the internet.**
