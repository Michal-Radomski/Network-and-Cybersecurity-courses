**DHCP (Dynamic Host Configuration Protocol)** is a network protocol that **automatically assigns IP addresses and network
settings** to devices on a network.

### ⚡ Key points

- Eliminates the need to manually configure IP addresses
- Works on local networks (home, office, Wi-Fi)
- Assigns settings like IP address, subnet mask, gateway, DNS

### 🔄 How it works (DORA process)

1. **Discover** → Device asks: “Any DHCP server available?”
2. **Offer** → Server առաջարկs an IP address
3. **Request** → Device requests that specific IP
4. **Acknowledge** → Server confirms and assigns it

### 🧩 What DHCP provides

- IP address (e.g., 192.168.1.10)
- Subnet mask
- Default gateway (router)
- DNS servers

### 🧠 Simple example

When you connect your phone to Wi-Fi, DHCP automatically gives it an IP address so it can access the internet.

👉 In short: **DHCP automatically configures devices so they can communicate on a network.** **DHCP (Dynamic Host
