**Vulnerability Scanning** is the automated process of proactively identifying, evaluating, and reporting security weaknesses
in your computer systems, networks, or applications.

Think of it like a digital "home inspection." A vulnerability scanner goes through every "window and door" (ports and
services) of your system and checks if the "locks" (software versions and configurations) are broken or outdated. It then
compares what it finds against a massive database of known flaws (called **CVEs** or Common Vulnerabilities and Exposures).

---

### How Vulnerability Scanning Works

1.  **Detection:** The tool identifies all active devices and services on the network.
2.  **Inventory:** It determines what operating systems and applications are running (e.g., Windows 11, Ubuntu 22.04, Apache
    web server).
3.  **Comparison:** It checks those versions against a database of thousands of known vulnerabilities.
4.  **Risk Assessment:** It assigns a severity score (usually **CVSS**) to each flaw so you know what to fix first.
5.  **Reporting:** It generates a list of "remediation steps" (usually "Update this software" or "Change this setting").

---

### Top Tools for Windows and Ubuntu

Vulnerability scanners generally fall into two categories: **Network Scanners** (which scan from the outside) and
**Host/Agent Scanners** (which run directly on the machine).

#### 1. Nessus (The Industry Standard)

- **OS:** Windows, Ubuntu, and macOS.
- **Details:** Nessus is the most popular scanner in the world. It is highly accurate and has a massive library of plugins.
- **Best for:** Professionals and students. There is a **"Nessus Essentials"** version that is free for home use (up to 16 IP
  addresses).

#### 2. OpenVAS (The Open Source Powerhouse)

- **OS:** Primarily Linux/Ubuntu (can scan Windows machines over the network).
- **Details:** If you want a free, enterprise-grade tool without IP limits, OpenVAS (part of the Greenbone Vulnerability
  Management suite) is the go-to choice.
- **Best for:** Ubuntu users who are comfortable with the command line or setting up web interfaces.

#### 3. Wazuh (Endpoint Detection & Response)

- **OS:** Windows and Ubuntu (Agent-based).
- **Details:** Wazuh is an all-in-one security platform. You install a small "agent" on your Windows or Ubuntu machine, and
  it constantly monitors for vulnerabilities, missing patches, and even active attacks.
- **Best for:** Continuous monitoring rather than a "one-off" scan.

#### 4. Nmap with Vulners Script (The Lightweight Choice)

- **OS:** Windows (via Nmap GUI/Zenmap) and Ubuntu (Terminal).
- **Details:** Nmap is famous for network mapping, but if you use the `--script vulners` flag, it transforms into a basic
  vulnerability scanner.
- **Best for:** Quick, fast checks without installing heavy software.

### A Critical Distinction

It’s important to remember that **Vulnerability Scanning** is not the same as **Penetration Testing**.

- **Scanning** identifies the hole in the fence.
- **Penetration Testing** involves a human actually trying to climb through that hole to see how far they can get.
