**ICMP (Internet Control Message Protocol)** is a network protocol used for **sending error messages and diagnostic
information** between devices on an IP network.

### ⚡ Key points

- Works alongside IP (not used for regular data transfer)
- Helps diagnose network issues
- Used by tools like **ping** and **traceroute**

### 🧩 What it does

- Reports errors (e.g., destination unreachable)
- Checks connectivity between devices
- Measures network latency

### 🧠 Simple example

When you run `ping`, ICMP sends a request to another device and waits for a reply to confirm it’s reachable.

👉 In short: **ICMP helps devices communicate about network status and problems.**
