Malware is the broad "umbrella" term for any software designed to cause harm. Viruses, worms, and Trojans are specific types
of malware, categorized by how they spread and interact with systems. [1, 2, 3, 4]

## Types of Malicious Software

- Virus: A program that attaches itself to a legitimate file or application. It stays dormant until a human runs or opens the
  host file (like an email attachment), at which point it activates and attempts to infect other files.
- Worm: Standalone software that self-replicates and spreads automatically across networks without needing a host file or
  human action. Worms often exploit security vulnerabilities in operating systems to hop from one device to another.
- Trojan (Trojan Horse): Malware that disguises itself as legitimate or useful software to trick you into installing it.
  Unlike viruses or worms, Trojans do not self-replicate; they rely on social engineering to get inside, where they may
  create backdoors for hackers or steal sensitive data. [1, 4, 5, 6, 7, 8, 9, 10]

## Comparison at a Glance

| Feature [7, 11, 12, 13, 14] | Virus                       | Worm                         | Trojan                             |
| --------------------------- | --------------------------- | ---------------------------- | ---------------------------------- |
| Needs a Host?               | Yes (attaches to a file)    | No (standalone)              | No (standalone)                    |
| Self-Replicates?            | Yes                         | Yes (at high speeds)         | No                                 |
| User Action?                | Required (opening the file) | Not Required                 | Required (tricked into installing) |
| Main Threat                 | Corrupts or deletes data    | Consumes bandwidth/resources | Steals data/gives remote access    |
