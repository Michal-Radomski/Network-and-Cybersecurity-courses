In computer networking, ports are logical endpoints used to identify specific processes or services on a device. Protocols
are the sets of rules that define how data is exchanged. [1, 2, 3] Ports are divided into three main ranges: [4, 5, 6, 7]

- Well-Known Ports (0–1023): Reserved for standard, high-priority system services.
- Registered Ports (1024–49151): Used by specific applications like databases.
- Dynamic/Private Ports (49152–65535): Temporarily assigned for client connections. [4, 8, 9, 10]

## Key Transport Protocols

Most ports operate using one of two primary transport protocols: [7, 19, 20, 21]

- TCP (Transmission Control Protocol): Used for services where reliability is critical (like web browsing or email) because
  it ensures all data arrives in order.
- UDP (User Datagram Protocol): Used for speed-sensitive services where losing some data is acceptable (like video streaming
  or DNS queries). [7, 13, 18, 19]
