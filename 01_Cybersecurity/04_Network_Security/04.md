A MAC (Media Access Control) address is a unique hardware identifier assigned to a
[network interface card (NIC)](https://dnschecker.org/mac-lookup.php) during manufacturing. While an IP address serves as
your "postal address" on the internet, the MAC address is like your device's permanent "fingerprint" or "social security
number". [1, 2, 3, 4, 5]

## Key Characteristics

- Physical Address: It is often called a physical or burned-in address because it is hardcoded into the hardware.
- Layer 2 Operation: It functions at the Data Link Layer (Layer 2) of the OSI model, facilitating communication within a
  local network.
- Global Uniqueness: Every network-enabled device—including your phone, smart fridge, and laptop—has a unique MAC address.
- Interface-Specific: A single device can have multiple MAC addresses—one for its Ethernet port, one for Wi-Fi, and one for
  Bluetooth. [2, 3, 5, 6, 7, 8, 9]

## Structure and Format

A standard MAC address is a 48-bit (6-byte) number, typically represented as 12 hexadecimal characters (0–9, A–F). These are
usually grouped in pairs separated by colons or hyphens (e.g., 00:1A:2B:3C:4D:5E). [4, 8, 9, 10, 11]

- First 6 Characters (OUI): The [Organizationally Unique Identifier](https://liora.io/en/all-about-mac-address) identifies
  the manufacturer (e.g., Dell, Apple, or Cisco).
- Last 6 Characters: A unique serial number assigned by that specific manufacturer. [4, 8, 12]

## MAC vs. IP Address

| Feature [13, 14, 15, 16, 17] | MAC Address              | IP Address          |
| ---------------------------- | ------------------------ | ------------------- |
| Address Type                 | Physical/Hardware        | Logical/Software    |
| Permanence                   | Fixed (Hardcoded)        | Temporary (Dynamic) |
| Scope                        | Local Network (LAN) only | Global (Internet)   |
| OSI Layer                    | Layer 2 (Data Link)      | Layer 3 (Network)   |
