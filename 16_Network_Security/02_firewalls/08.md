## What is Stealth Mode?

In networking, **Stealth Mode** is a firewall security feature that prevents your computer from responding to unsolicited
network traffic, specifically **ping requests** (ICMP packets) and port scans.

Normally, when a device is connected to a network, it will respond to a ping request by saying, "Yes, I am here." If a hacker
is scanning a public Wi-Fi network for targets, these responses tell them that your computer is active and worth attacking.

When Stealth Mode is enabled:

- Your computer completely ignores these requests.
- To anyone scanning the network, your device appears as if it is turned off or doesn't exist at all.
- It drops the packets silently instead of sending back a "Connection Refused" message.

---

## How to Switch It On on Ubuntu

Ubuntu uses **UFW (Uncomplicated Firewall)** by default. To achieve "Stealth Mode" on Ubuntu, you need to configure UFW to
drop packet requests silently instead of rejecting them, and explicitly tell it to ignore ICMP (ping) traffic.

Here is how to do it step-by-step using the Terminal:

### Step 1: Make sure UFW is enabled

First, check if your firewall is active.

```bash
sudo ufw status

```

If it says `Status: inactive`, turn it on by running:

```bash
sudo ufw enable

```

### Step 2: Set the default policy to DROP

By default, UFW drops incoming traffic, but we want to ensure it is strictly dropping (ignoring) rather than rejecting (which
sends a response back).

```bash
sudo ufw default drop incoming

```

### Step 3: Block Ping (ICMP) Requests

To truly be in stealth mode, you need to edit the UFW configuration file to stop responding to pings.

1. Open the UFW configuration file in a text editor (like `nano`):

```bash
sudo nano /etc/ufw/before.rules

```

2. Scroll down through the file until you find a section that looks like this (usually near the top):

```text
# ok icmp codes for INPUT
-A ufw-before-input -p icmp --icmp-type echo-request -j ACCEPT

```

3. Change the word `ACCEPT` to `DROP` at the end of that line:

```text
-A ufw-before-input -p icmp --icmp-type echo-request -j DROP

```

4. Save and exit the file (In Nano, press `Ctrl + O`, then `Enter` to save, and `Ctrl + X` to exit).

### Step 4: Reload the Firewall

Apply the changes by reloading UFW:

```bash
sudo ufw reload

```

Your Ubuntu system is now in Stealth Mode. It will silently ignore incoming connection attempts and ping requests, making it
invisible to network scanners.
