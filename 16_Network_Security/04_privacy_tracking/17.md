Firefox utilizes internal pages (often called `about:` URIs) to provide built-in tools, settings, diagnostics, and hidden
features.

Here is a list of the most useful and commonly accessed `about:` pages in Firefox:

### Configuration and Customization

- **`about:config`** – The most powerful internal page. It contains a comprehensive database of hidden configuration settings
  and preferences. Changing settings here can alter browser behavior, performance, and UI elements.
- **`about:preferences`** – The standard settings and options menu where you manage privacy, security, theme, and general
  behavior.
- **`about:addons`** – The Add-ons Manager, where you can view, configure, enable, or disable extensions, themes, and
  plugins.
- **`about:profiles`** – Allows you to create, manage, and switch between multiple user profiles (each with its own
  bookmarks, history, and settings).

### Diagnostics and Performance

- **`about:memory`** – A detailed tool for monitoring memory usage. It allows you to see exactly how much RAM each tab and
  extension is using and includes a button to "Minimize memory usage" by forcing garbage collection.
- **`about:performance`** (or **`about:processes`**) – A built-in task manager that shows real-time CPU and memory usage for
  open tabs and active extensions, allowing you to close problematic tabs.
- **`about:support`** – The Troubleshooting Information page. It displays technical details about your installation (graphics
  driver, application binary path, active experiments) and features a "Refresh Firefox" button to reset the browser to its
  default state.
- **`about:crashes`** – Lists details and submission logs for recent browser crashes, which is helpful for troubleshooting
  stability issues.
- **`about:networking`** – Provides detailed insights into current network connections, DNS cache, sockets, and webRTC
  traffic.

### Information and Statistics

- **`about:about`** – A master index that lists every single `about:` page available in your specific version of Firefox.
- **`about:telemetry`** – Shows the performance and usage data that Firefox collects and sends back to Mozilla (if telemetry
  is enabled) to help improve the browser.
- **`about:downloads`** – Opens the dedicated download history library panel.
- **`about:cache`** – Displays information about the memory and disk cache, showing where temporary files are stored.
- **`about:license`** – Displays the legal licenses for the open-source software and pieces of code used to build Firefox.

### Miscellaneous & Easter Eggs

- **`about:robots`** – A classic Firefox Easter egg containing humorous pop-culture references to robots.
- **`about:mozilla`** – Displays a cryptic, apocalyptic quote from the fictional "Book of Mozilla," a long-running joke in
  the history of Netscape and Firefox.
