**VeraCrypt** is a popular, free, and open-source utility used for on-the-fly data encryption. It is a highly secure
successor to the discontinued TrueCrypt project.

With VeraCrypt, you can:

- **Create a virtual encrypted disk inside a file** (known as a "file container") and mount it like a regular physical drive.
- **Encrypt an entire partition or storage device**, such as a USB flash drive or external hard drive.
- **Work cross-platform**, meaning you can seamlessly open and edit your encrypted files on Ubuntu, Windows, and macOS.

---

## How to Install VeraCrypt on Ubuntu

There are two primary ways to install VeraCrypt on Ubuntu: **via a PPA** (the easiest way to keep it updated) or by
**downloading the official installer package** directly from the website.

### Option 1: Install via PPA (Recommended & Easiest)

Using a Personal Package Archive (PPA) is highly convenient because it allows VeraCrypt to be updated automatically alongside
your other system packages when you run your normal system updates.

1. Open your terminal (`Ctrl` + `Alt` + `T`).
2. Add the well-known `unit193` encryption repository to your system:

```bash
sudo add-apt-repository ppa:unit193/encryption -y

```

3. Update your package list:

```bash
sudo apt update

```

4. Install VeraCrypt:

```bash
sudo apt install veracrypt -y

```

---

### Option 2: Download the Official Package (Manual)

If you prefer not to use third-party repositories, you can install the official Debian package directly from the developers.

1. Go to the official [VeraCrypt Downloads page](https://www.veracrypt.fr/en/Downloads.html).
2. Scroll down to the Linux section and download the `.deb` package matching your Ubuntu version (GUI version is recommended
   for desktops, e.g., `veracrypt-x.xx.xx-Ubuntu-xx.xx-amd64.deb`).
3. Alternatively, you can download it via the terminal using `wget` (be sure to replace the link below with the newest
   version from their page):

```bash
wget https://launchpad.net/veracrypt/trunk/1.26.24/+download/veracrypt-1.26.24-Ubuntu-24.04-amd64.deb

```

4. Install the downloaded package:

```bash
sudo dpkg -i veracrypt-*.deb

```

5. If there are any missing dependency errors, fix them by running:

```bash
sudo apt install -f

```

---

## Launching VeraCrypt

Once installed, you can launch the application in one of two ways:

- **GUI:** Open your Applications menu, search for **VeraCrypt**, and click the icon to launch the graphical wizard.
- **Terminal:** You can launch the GUI or use its command-line features by typing:

```bash
veracrypt

```
